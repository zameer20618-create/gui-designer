🐙 How to Host on GitHub Pages
GitHub Pages is free and hosts static HTML files publicly — perfect for this tool.
Step 1 — Create a GitHub account
Go to github.com and sign up for a free account if you don't have one.
Step 2 — Create a new repository
Click New → name it something like roblox-gui-studio → set to Public → click Create repository.
Step 3 — Upload your file
In your new repo, click Add file → Upload files
Upload roblox-gui-studio.html
⚠️ Rename it to index.html before uploading (or rename after). GitHub Pages serves index.html as the homepage.
Step 4 — Enable GitHub Pages
Go to your repo → Settings tab → scroll down to Pages (left sidebar)
Under Source, select Deploy from a branch
Branch: main / (root) → click Save
Step 5 — Access your site
After ~1–2 minutes, your site will be live at:
https://YOUR-USERNAME.github.io/roblox-gui-studio/
💡 Pro Tips
• The AI feature requires the Anthropic API. For the AI to work when self-hosted, you need to add an API key. Since this is a client-side app, use a proxy or consider Cloudflare Workers to keep your key secret.
• All other features (canvas, code gen, sounds, templates) work offline with zero API needed.
• To update: just re-upload index.html and GitHub Pages auto-redeploys.
